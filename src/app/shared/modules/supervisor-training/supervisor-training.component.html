Please populate your list of supervisors who have been trained using one of the following options. Each option updates your list in real-time. Clicking save simply marks this audit step as completed. If your list is already current, you can click Save to mark this step as completed. Option 1 - You can use the grid to maintain your list one employee at a time (this is often the best option if you only have a few supervisors on your list). Option 2 - You can also upload your list using the template available below. This process updates your list in real-time. Clicking save simply marks this audit step as completed. If your list is already current, you can click Save to mark this step as completed.

You can replace your current supervisor training list with one that you upload. This is helpful if you have many supervisors and require many modifications to your list. There are a few pre-requisites: you must use a template that is provided above and the templates are in Microsoft Excel format and therefore should be modified using Microsoft Excel. Using another software application (Libre Office for example) to modify the Excel documents may not work.

To import a new Supervisor Training List please use the following procedure:
  1. Download the template .xls document or your current supervisor training list in .xls fomat using the buttons above. Using any other file to upload your data will cause your import to fail.
  2. Modify the document according to the instructions contained within the document.
  3. Save the file on your computer and close the file. (uploading open files will fail)
  4. Click Select to locate the file and browse to the file you want to upload.
  5. Click the Import button to process the uploaded file.
  6. If the data in your file is valid, your current Supervisor Training list will be replaced with the data you imported.
  7. If there are errors in the file you uploaded, your Supervisor Training list will not be replaced and you will be notified of invalid entries.
